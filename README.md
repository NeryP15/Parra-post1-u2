# Auditoría Heurística - Nequi 

## 1. Descripción del Proyecto
Este proyecto consiste en una auditoría de usabilidad sistemática aplicada a la aplicación móvil **Nequi** (Plataforma: Android/iOS), utilizando las 10 heurísticas de Jakob Nielsen.

 ## 2. Alcance de la Evaluación
Se han seleccionado los siguientes 3 flujos críticos para la auditoría:

**Flujo A: Inicio de sesión y consulta de saldo.**
  Justificación:Es el punto de entrada principal y debe ser rápido y libre de errores para generar confianza.
**Flujo B: Envío de dinero a un contacto.**
  Justificación: Es la función núcleo de la app; cualquier fricción aquí afecta directamente la utilidad del servicio.
**Flujo C: Configuración de seguridad (Ajustes).**
  Justificación: Crucial para que el usuario sienta control sobre la protección de sus fondos.
